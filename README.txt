Pré requis: Installation de l'extension de Google Chrome "Postman"

Pour exécuter le projet suivre les étapes suivante:

	1. Decomprésser les archives
	2. Ouvrir InteliJ
	3. Cliquer sur Importe un projet
	4. Selectionner le fichier pom.xml du projet à importer
	5. Exécuter chaque projet
	6. Ouvrir "Postman"
	7. Ecrire les requetes à effectuer dans la barre de Postman et ne pas oublier de sélectionner le type de requête à effectuer (GET, POST, etc...)
